### 📍Functions

| feature        | description                                                    |
| -------------- | -------------------------------------------------------------- |
| 계산기 기능    | 입력받은 값 디스플레이에 표시 및 연산을 통한 결과값 출력       |
| alert 컴포넌트 | 잘못된 연산 과정 입력 후 결과 버튼 눌렀을 경우, 경고 창 도출   |
| 세뱃돈 버튼    | 세뱃돈으로 많이 받는 만원, 5만원, 10만원 단위 버튼 별도로 생성 |
